# 经济学模拟器设计文档

## 1. 项目概述

### 1.1 目标
构建一个中等复杂度的经济学模拟网站，核心关注点是**加班时长**和**加班率**对经济的影响机制，支持动态观察多期经济演化。

### 1.2 核心特性
- 单一经济体模拟
- 时间步长可配置（年度/季度）
- 完整的加班影响机制（劳动投入、工资收入、疲劳效应、劳动力损耗）
- 外贸参数内生决定
- 仪表盘式交互界面
- 支持部署到 GitHub Pages（免费托管）

### 1.3 技术栈
- **前端框架**: React + TypeScript
- **图表库**: Recharts
- **状态管理**: Zustand
- **部署**: GitHub Pages

## 2. 经济模型设计

### 2.1 核心生产函数

采用**柯布-道格拉斯生产函数**：

```
Y = A × K^α × L^(1-α)
```

| 符号 | 含义 | 说明 |
|------|------|------|
| Y | GDP产出 | 当期总产出 |
| A | 全要素生产率 | 技术水平 |
| K | 资本存量 | 累计资本 |
| L | 有效劳动投入 | 见下方公式 |
| α | 资本产出弹性 | 默认0.35 |

### 2.2 劳动市场与加班机制（核心）

**有效劳动投入：**
```
L = N × h × η
```

| 符号 | 含义 |
|------|------|
| N | 就业人数 |
| h | 人均工作时长（含加班） |
| η | 劳动生产率（受疲劳影响） |

**加班参数定义：**
```
h = h₀ + overtime        // 实际工时 = 正常工时 + 加班时长
overtime_rate = 用户设定  // 加班率：有多少比例的人在加班
```

**疲劳效应（生产率下降）：**
```
η = η₀ × (1 - fatigue_factor)

fatigue_factor = β × max(0, h - h₀) / h₀
```
- 工作时长超过正常值后，生产率按比例下降
- β 为疲劳敏感系数（用户可调）

**加班工资效应：**
```
wage = wage_base × (1 + overtime × overtime_premium × overtime_rate)
```
- 加班费率默认 1.5倍

**劳动力留存（长期效应）：**
```
N_next = N × (1 - attrition)
attrition = base_attrition + γ × cumulative_overtime_stress
```
- 长期高加班 → 劳动力流失

**失业率模型：**
```
失业率 = 1 - (就业人数 / 劳动力人口)

劳动力人口 = 总人口 × 劳动参与率

就业需求 = Y / (h × η)    // 生产一定产出所需工人数
```
加班增加 → 人均工时h增加 → 就业需求减少 → 失业率上升（挤出效应）

### 2.3 消费与投资

```
C = c × (wage × N)      // 消费 = 消费倾向 × 劳动总收入
I = s × Y               // 投资 = 储蓄率 × 产出
K_next = K × (1 - δ) + I // 资本累积（δ为折旧率）
```

### 2.4 外贸模块（内生决定）

**出口增长率内生机制：**
```
出口增长率 = 基础增长率 + 竞争力效应 + 产能效应

其中：
- 竞争力效应 = λ₁ × ΔA/A    // 技术进步 → 产品竞争力提升
- 产能效应 = λ₂ × ΔY/Y      // 产出增长 → 出口供给能力增强

EX_t = EX_{t-1} × (1 + 出口增长率)
```

**进口倾向内生机制：**
```
进口倾向 = 基础倾向 + 收入效应

其中：
- 收入效应 = θ × (人均收入增长 - 基准增长)

IM_t = 进口倾向_t × C_t    // 进口与消费挂钩
```

| 参数 | 含义 | 默认值 |
|------|------|--------|
| λ₁ | 技术对出口的弹性 | 0.3 |
| λ₂ | 产能对出口的弹性 | 0.2 |
| θ | 收入对进口的敏感度 | 0.1 |

### 2.5 GDP核算

```
Y = C + I + G + NX
```
（G 为政府支出，可设为固定值或比例）

### 2.6 模型运行逻辑

```
每个时间步：
1. 读取参数（加班时长、加班率、进出口增长率等）
2. 计算有效劳动 L（含疲劳效应）
3. 计算产出 Y
4. 分配收入（工资、利润）
5. 计算消费、投资、净出口
6. 更新资本存量 K
7. 更新劳动力 N（含留存效应）
8. 计算失业率
9. 记录所有指标，进入下一期
```

### 3.2 交互设计

| 操作 | 行为 |
|------|------|
| **调节参数** | 实时更新图表（防抖500ms后重新计算） |
| **开始/暂停** | 动画播放模拟过程，每期间隔可配置 |
| **重置** | 恢复所有参数和结果到初始状态 |
| **保存** | 将当前参数和结果存入 LocalStorage |
| **导出** | 下载 CSV 文件（包含所有期数数据） |
| **图表交互** | 鼠标悬停显示数值，可缩放时间范围 |
| **图表折叠/展开** | 点击图表标题栏可收起/展开，收起状态仅显示标题 |
| **全部展开/收起** | 提供一键操作按钮 |
| **折叠状态记忆** | 保存到 LocalStorage，下次访问保持用户偏好 |
